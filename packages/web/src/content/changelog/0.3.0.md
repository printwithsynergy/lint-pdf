### Added

- **Custom Flight Plans** — Growth, Scale, and Enterprise plans can now create custom preflight profiles with specific Inspections and thresholds via `POST /api/v1/flight-plans`.
- **Tower Alerts** — Webhook callbacks for real-time notification when files finish processing. Register endpoints at `POST /api/v1/tower-alerts` to receive `checkin.arrived`, `checkin.grounded`, and `checkin.clear` events.
- **Packaging Flight Plan** — New built-in Flight Plan with 210 Inspections including barcode grading per ISO 15416.

### Changed

- Inspection count increased from 196 to 250+ with new barcode, page geometry, and compliance checks.
- Flight Log response now includes a `verdict` field (`clear-to-fly` or `grounded`) for quick pass/fail evaluation.

### Fixed

- Resolved an issue where spot color detection could miss colors defined in nested form XObjects.
- Fixed ink coverage calculation for pages with transparency groups.
