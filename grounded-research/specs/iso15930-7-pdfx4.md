## SECTION 1: SCOPE AND CONFORMANCE OVERVIEW

### 1.1 Scope Definition
**From ISO 15930-7:2010 Clause 1**

This standard specifies the use of PDF Version 1.6 for dissemination of digital data intended for print reproduction.

**Two Conformance Levels**:
- **PDF/X-4**: Complete exchange of printing data (all necessary elements for final print reproduction contained within the file)
- **PDF/X-4p**: Partial exchange with external profile reference (required ICC profile supplied externally and unambiguously identified)

**Supported Capabilities**:
- Colour-managed, CMYK, gray, RGB or spot colour data
- PDF transparency (major differentiator from PDF/X-1a)
- Optional content (layers) for regional versioning
- PDF 1.6 features

### 1.2 Conformance Levels Comparison

## SECTION 2: KEY DIFFERENCES - PDF/X-4 vs. PDF/X-1a

### 2.1 Major Enhancements

**PDF/X-4 introduces over PDF/X-1a**:
1. Transparency support - allows blend modes, soft masks, extended graphics state
2. Optional content (layers) - enables regional versioning
3. Colour management - ICC profiles with CMYK, RGB, Gray support
4. PDF 1.6 features - includes transparency operators and optional content streams
5. Greater flexibility while maintaining print safety

### 2.2 Features Allowed (First Time in PDF/X)
- Transparency and blend modes
- Optional content (OCG - Optional Content Group)
- External ICC profile reference (PDF/X-4p variant)

## SECTION 6: CRITICAL CONFORMANCE CRITERIA

### Minimum Requirements for PDF/X-4 Validity

A file is **NOT** PDF/X-4 conformant unless ALL of the following CRITICAL requirements are met:

1. **PDFX4-001**: PDF 1.6 version header
2. **PDFX4-003**: GTS_PDFXVersion = "PDF/X-4" in XMP metadata
3. **PDFX4-005 to PDFX4-010**: Complete OutputIntent with embedded ICC profile
4. **PDFX4-013, PDFX4-015**: Color spaces limited to DeviceGray, DeviceRGB, DeviceCMYK, or ICCBased
5. **PDFX4-022**: All fonts embedded
6. **PDFX4-032, PDFX4-036**: Metadata and Info dictionaries present
7. **PDFX4-042, PDFX4-044**: TrimBox and BleedBox defined
8. **PDFX4-054**: No encryption
9. **PDFX4-059**: No JavaScript
10. **PDFX4-089**: All resources embedded (for PDF/X-4)
